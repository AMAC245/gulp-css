# gulp-css
Using Gulp.js for CSS tasks


## Build production CSS
Linux/Mac OS:

```bash
NODE_ENV=production
gulp css
```

(or inline `NODE_ENV=production gulp build`)

Windows Powershell:

```powershell
$env:NODE_ENV="production"
gulp css
```

Windows legacy command line:

```cmd
set NODE_ENV=production
gulp css
```
